【读书笔记01】存在的目的是为了寻求安乐

发表于 2022-12-26 分类于 所思所想

（一）承认痛苦的普遍性

（1）缺乏对痛苦的包容和忍耐，令我们脆弱不堪，打击、挫败接二连三，生活真的变得比较惨。反过来，如果我们认为生活中有痛苦是正常的，人生本来如此，我们就能更好地集中精力处理问题本身，而不是无谓地纠缠在愤愤不平的情绪中。
（2）如果痛苦、尴尬在所难免，我们最好让自己有所准备。这样做的好处是，虽然该面对的问题还得面对，该经历的痛还得经历，我们却不再那样感到苦，不必再承受额外的恐惧和焦虑。
（3）寂天菩萨曾说：“问题若有办法解决，就不必担心；若没办法解决，担心也没有用。”
（4）只要有这个身体在，我们就必定经历衰老、病痛、死亡；只要心里还有贪执、嗔恨、困惑、傲慢，我们就必定感受痛苦。
解：承认痛苦的普遍性，而不是把痛苦纯粹当作一种负面经历。好像找到了自己痛苦的来源。我总是在想发设法地去避免痛苦，或者认为痛苦是一种失败的表现，要是自己能力足够，一切都摆得平，就不会有痛苦。回顾自己在哪些时刻曾感受到痛苦：学业上没有取得理想成绩时；因为感受到努力并没有带来成功而痛苦。这种痛苦来源于预设了取得好成绩的时刻以及心绪变化，而从未预设过要是失败了会是怎样的感受，潜意识里并不承认或者接受失败。所以，哪有人一直成功，接受自己，接纳自己。向内不断迭代更新，然后越来越好。第二是与他人比较而感受到的痛苦。这一点暂时还无解，因为人与人之间必定会有联系，不能依靠封闭自我来避免比较，这反而是一种更为痛苦的方法。

（二）无我有助于减弱对外物的贪执

（1）比如生病了我们一般会说：我不舒服！我很痛！我很惨！“这时候如果我们提醒自己：没有我，只是这个肉体的某些部分、某些功能出了问题，不舒服、疼痛也只是一时的感受，而感受随时在变化。仅仅是知道没有一个实存的我在生病、在受苦，都会令心里的焦虑、恐惧放松很多。
（2）我们把自己的攀缘心、分别念、错觉投射在物体上，执著贵贱美丑，枉受痛苦。

（三）相信因果就是诸恶莫作、众善奉行、自净其意

（1）我们存在的目的就是为了寻求安乐，为了获得满足感。
（2）对修行人而言，生病使我们更真切地体会到众生的痛苦和他们对健康快乐的渴求。平常认为自己理所当然就该拥有的东西，像能看见这世界的五彩缤纷、能听见鸟语、能闻见花香、能尝到酸甜苦辣、能感受风净凉恬、能哭、能笑、能跑、能跳，乃至饿了能吃、困了睡觉……这都不是随时想有就能有的。身体健康的人往往忽视健康的可贵，不知道没有病痛的身体多么自由轻松，令人羡慕。
（3）人不可避免地要在短暂的一生中经历忧患变迁，他本能会对痛苦生起畏惧、躲避之心，希求安乐。
（4）人生充满起伏变化，很多时候自己的想法、计划都无法实现，但不管在什么情况下，都要提醒自己：暇满难得，今已得，人寿无常，死期不定，才不辜负这珍宝人生。

圣诞快乐，希望你是快乐的

发表于 2022-12-25 更新于 2022-12-26 分类于 所思所想

若无闲事挂心头，便是人间好时节

恍惚之间假期过去了一大半，这半个月新加坡一直在雨季，有时淅淅沥沥，大部分时候是雷雨，持续不了多久的阵雨伴随着雷。每天无所事事的睡觉，找饭吃，报复性地刷着手机，到最后拿着手机都不知道应该点开哪个软件。就算是没有意思，也能在小红书浪费掉一两个小时。这样不好，却也找不出第二种生活方式来度过假期，如何对抗生活的无意义感，这个问题我也思考了很久。无意义就无意义吧，闲暇时光也不可多得，不出意外将是最后一个寒假，学生时代的特权。
吾日三省吾身，最近也发现了自身的很多问题。首先，松弛不是敷衍。比起高中时候，身上的冲劲与韧性少了一半。想做好的事情太多了，但是没有哪件事做到了百分之百，没有百分之百的投入与坚持，哪来百分之百的结果。一分耕耘一分收获虽然不是绝对的真理，但希望每次能有一个坚定的信念与高标准来要求自己。不可否认的是人与人之间的差距来源于平时的一点一滴，时刻反省，不能人云亦云。第二，放下傲慢与偏见，相信人心本善。无论是环境还是网络带来的人与人之间的偏见，都带着自身以偏概全的想法，何况周围的人都在过着属于自己的精彩生活，接受人的多元化，不对他人的生活与人格做过多的评价。一句话，少对他人评头论足，平等地看待、对待每一个人。第三，认同自己，接纳自己。最近找到了适合自己的妆造，皮肤状态变好，朋友圈也在积极营业。一切的自信来源于自我认同感。生活不是一定要去完成什么，让自己开心最重要。活在当下，不留恋过去，不畏惧未来。
今天就到这里吧，希望下一次能有更有趣的事情可以分享。
晚安，世界！

我的第一篇博客

发表于 2022-12-18 更新于 2022-12-23 分类于 所思所想

源

前段时间考试月，人总是在繁忙的时候才会想起思考。有两周在UT过着昏沉的日子，早起看日出，比太阳的出勤率还高。某一天午饭的时间打开了《十三邀》，黄灯老师的这一期让人印象深刻，工业废水四个字直击人心，“空心人，带着塑料和电子产品的气味。”

是的，太久没有提起过笔，太久没有进行过有效写作，以至于当我想表达内心所思所想的时候，脑子里无法蹦出词句。静心看书也变成一件极其困难的事，月初下载的《次第花开》至今都还停留在第一页。不夸张的说，电子产品吞噬掉了我的所以碎片时间。写下你想要写的，那才是最要紧的。
“你必须照亮自己灵魂的深刻与浅薄，虚荣与宽宏，说出你的美貌和平庸对你意味着什么，以及你与这个变动不休的世界有何关系。“
我知道没有什么东西是长久的，我想不停地记录，所思，所感，算是作为“人”的一点存在感。

愿

不要麻木，去思考。
